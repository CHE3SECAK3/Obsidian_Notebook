# Power
- *Work per unit time*

- Units: *Watt* -> $W$

- $P = \large \frac{dW}{dt}$, where:

	- $P$ : Power ($W$)
	- $W$ : Energy ($J$)
	- $t$ : Time ($s$)

- $P = \frac{dW}{dt} = \left( \frac{dW}{\cancel{dQ}} \right) \left( \frac{\cancel{dQ}}{dt} \right) = VI$, where:
	- $P$ = Power ($W$)
	- $V$ = Voltage ($V$)
	- $I$ = Current ($A$)

- If $P = VI$ and $V = IR$, then:
	-  $P = I^2R$
	- $P = \large \frac{V^2}{R}$
