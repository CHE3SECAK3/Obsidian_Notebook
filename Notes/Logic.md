# Logic
- *The language of reasoning*

## Logical Operators
| Operator     | Symbol  |
|:------------ |:------- |
| Negation     | $\neg$  |
| Conjunction  | $\land$ |
| Disjunction  | $\lor$  |
| Exclusive OR | $\oplus$     |


# Proposition
- *A statement that is either true or false*
- Propositions have a **truth value** that could evaluate to *True* or *False*
