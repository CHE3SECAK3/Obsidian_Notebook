## Logical Operators in Precedence Order
| Operator               |   Symbol   |
|:---------------------- |:----------:|
| Universal Quantifier   | $\forall$  |
| Existential Quantifier | $\exists$  |
| Uniqueness Quantifier  | $\exists!$ |
| Negation               |   $\neg$   |
| Exclusive OR           |  $\oplus$  |
| Conjunction            |  $\land$   |
| Disjunction            |   $\lor$   |
| Conditional            | $\implies$ |
| Biconditional          |   $\iff$   |
